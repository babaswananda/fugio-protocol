Quick Example

import { FugioSDK } from '@fugio/sdk';

// Initialize SDK
const fugio = new FugioSDK({
  network: 'testnet',
  apiKey: 'your-api-key'
});

// Deploy a trading agent
const agent = await fugio.agents.deploy({
  name: 'MyTradingBot',
  type: 'defi-trader',
  config: {
    maxPositionSize: 1000,
    riskTolerance: 0.7,
    strategies: ['arbitrage', 'yield-farming']
  }
});

// Execute a mission
const result = await agent.execute({
  action: 'analyze-pool',
  params: {
    protocol: 'uniswap-v3',
    pair: 'ETH/USDC'
  }
});

console.log('Analysis:', result.data);

// Monitor earnings
agent.on('earnings', (earnings) => {
  console.log(`Earned: ${earnings.amount} FUGIO`);
});

API Reference

Core Endpoints

POST /api/v1/agents CREATE

Deploy a new AI agent instance with custom configuration

Request Body:
{
  "name": "string",
  "type": "defi-trader|content-creator|researcher",
  "config": { ... },
  "stake": "number"
}
GET /api/v1/agents/{id} READ

Retrieve agent status, configuration, and performance metrics

Response:
{
  "id": "string",
  "status": "active|paused|stopped",
  "earnings": "number",
  "performance": { ... }
}
POST /api/v1/missions EXECUTE

Submit a mission for agent execution with parameters

Request Body:
{
  "agentId": "string",
  "type": "analyze|trade|create|research",
  "params": { ... },
  "priority": "low|normal|high"
}

WebSocket Events

Real-time Updates

// Connect to WebSocket
const ws = new WebSocket('wss://api.fugio.io/ws');

// Listen for agent events
ws.on('agent.earnings', (data) => {
  console.log('New earnings:', data);
});

ws.on('agent.status', (data) => {
  console.log('Status update:', data);
});

ws.on('mission.completed', (data) => {
  console.log('Mission done:', data);
});

Event Types

agent.earnings New FUGIO earned
agent.status Status change
mission.started Mission began
mission.completed Mission finished
network.alert Network issues
