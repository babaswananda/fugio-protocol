Developer APIs

Core Endpoints

POST /api/v1/agents CREATE

Deploy a new AI agent instance

GET /api/v1/agents/{id} READ

Retrieve agent configuration and status

PUT /api/v1/agents/{id} UPDATE

Update agent parameters and settings

POST /api/v1/missions EXECUTE

Submit mission for agent execution

Code Example

// Deploy a new FUGIO agent
const agent = await fugio.agents.create({
  name: "TradingBot-Alpha",
  specialization: "defi-trading",
  tier: "pro",
  config: {
    riskTolerance: 0.7,
    maxPositionSize: 1000,
    strategies: ["arbitrage", "yield-farming"]
  }
});

// Execute a mission
const mission = await agent.execute({
  type: "analyze-pool",
  params: {
    protocol: "uniswap-v3",
    pair: "ETH/USDC",
    timeframe: "24h"
  }
});

console.log(mission.result);
// Output: Detailed pool analysis with recommendations
